📡 Intel Swarm Public API

Free to use. No auth required. All responses are JSON. Base URL: https://intel-swarm.vercel.app

All findings for a date
/api/v1/findings?date=2026-03-05&domain=war
Params: date (YYYY-MM-DD), domain (e.g. war, crypto), lang (en/zh), limit (max 500)
Ticker-filtered findings
/api/v1/findings?ticker=BTC
Returns all findings where BTC is a relevant asset tag. Works with: BTC, ETH, SOL, OIL, GOLD, DXY, SPX, NVDA…
Full feed (score-sorted)
/api/v1/feed
All findings across all domains, sorted by intelligence score. Ideal for trading dashboards.
Daily brief
/api/v1/brief
AI synthesis + chief-of-staff brief as plain text. Good for summarization pipelines.
Search
/api/v1/search?q=iran+oil
Full-text search across all findings. Returns matching items with domain, tickers, score.
Domain list
/api/v1/domains
All 19 research domains with metadata.
RSS feed
/rss
Global RSS feed. Domain-specific: /rss/war, /rss/crypto, etc.
