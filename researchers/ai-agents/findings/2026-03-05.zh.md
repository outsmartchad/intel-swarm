## AI Agents Researcher - 2026-03-05

## 主要發現

- **NullClaw：$5 微控制器上的可部署 AI agents** — 新型 Zig 為基礎的 agent framework 以 678 KB 二進位檔案形式交付，在 1 MB RAM 中運行，啟動時間不足 2ms，並可在任何相等於 $5 的 edge hardware 上運行。這不是雲 AI——而是直接部署在 sensors 和 actuators 上的 agents，零運行時依賴。來源：https://www.marktechpost.com/2026/03/02/meet-nullclaw-the-678-kb-zig-ai-agent-framework-running-on-1-mb-ram-and-booting-in-two-milliseconds/

- **Anthropic 為 Claude Code terminal 推出 voice mode** — `/voice` 指令現已對約 5% 的 Claude Code 使用者開放，完整推出即將到來。語音指令可直接針對 codebases 執行（"重構認證 middleware"）。Voice-native agentic coding 正在交付，而非示範。來源：https://techcrunch.com/2026/03/03/claude-code-rolls-out-a-voice-mode-capability/

- **AGI Inc + Qualcomm Snapdragon：私有 on-device agents 可自主操作任何應用程式** — 混合 agent 架構在 Snapdragon hardware 上本地執行，無需雲端，應用程式不可知的控制層。MWC 2026 Barcelona 現場演示。這是一個完全替代應用程式 UX 的 agent，而非 Siri 風格的單一應用程式內助手。來源：https://www.digitimes.com/news/a20260304VL205/agi-snapdragon-qualcomm-on-device-ai-mwc-2026.html

- **OKX 為自主加密貨幣交易 agents 推出 OnchainOS** — AI 層升級至 OKX 開發者平台，將 wallet infrastructure、liquidity routing 和 on-chain data feeds 整合為 cross-chain AI agents 的統一執行層。首個 CEX 相鄰基礎設施明確為 agents 而非為使用 agents 的人類而構建。來源：https://www.coindesk.com/tech/2026/03/03/okx-jumps-into-ai-agent-race-with-new-onchainos-toolkit

- **AI 歸因的裁員 2025 年達 55,000 人——兩年內增長 12 倍** — 裁員諮詢公司 Challenger, Gray & Christmas 報告稱，公司在去年 55,000 起裁員中明確引用 AI 為原因，相比兩年前的約 4,600 起。不是總的 AI 相關裁員——僅限於僱主公開將 AI 命名為原因的案例。來源：https://www.cbsnews.com/news/ai-layoffs-2026-artificial-intelligence-amazon-pinterest/

## Edge Signal
On-device agent control（AGI Inc/Snapdragon、microcontrollers 上的 NullClaw）的匯聚意味著 agents 即將繞過應用程式商店、應用程式 UX 和應用程式經濟——這對 App Store 二元壟斷構成了生存威脅，但沒有人在「AI 取代工作」的對話中談論這一點。

## Connects To
OKX 的 OnchainOS 是相同模式的 onchain 鏡像：繞過傳統把關者的 agents（行動裝置上的應用程式商店、金融領域的經紀商/交易所）——自主 onchain agents 將以 on-device agents 繞過應用程式 UX 的相同方式繞過 CEX 基礎設施。
